# 快照指标说明：风险检测

本文用于解释 DangInvest `Snapshot API` 返回的 `dang.risk_checks` 字段，方便前端展示“来源/节选”并生成站内文章内链。

> 重要说明：以下阈值均为**工程化快速提示**，用于筛选与提醒，**不等同于投资建议**。不同公司、不同行业、不同会计口径下，指标可能存在不可比或需要额外解释的情况。

## 存贷双高（工程化）

存贷双高(工程化)：现金占比>25% 且 负债占比>35%。

解释：
- 现金占比 = `cash_and_equivalents / total_assets`
- 负债占比 = `interest_bearing_debt / total_assets`（这里用“有息负债/资产”作为快速代理）

## 商誉风险（工程化）

商誉风险(工程化)：商誉/资产>20% 提示，>35% 高风险。

解释：
- 商誉/资产 = `goodwill / total_assets`
- 该指标仅用于提示“并购形成的无形资产占比是否偏高”，并不能单独推导公司真实盈利质量或资产可回收性。
